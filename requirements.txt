1.2 Nginx Configuration (Copy-Paste Ready)

nginx.conf
server {
    listen 80;
    server_name dentaltripchina.com www.dentaltripchina.com;
    return 301 https://dentaltripchina.com$request_uri;
}

server {
    listen 443 ssl http2;
    server_name www.dentaltripchina.com;
    ssl_certificate     /etc/letsencrypt/live/dentaltripchina.com/fullchain.pem;
    ssl_certificate_key /etc/letsencrypt/live/dentaltripchina.com/privkey.pem;
    return 301 https://dentaltripchina.com$request_uri;
}

server {
    listen 443 ssl http2;
    server_name dentaltripchina.com;
    root /var/www/dentaltripchina/public;

    ssl_certificate     /etc/letsencrypt/live/dentaltripchina.com/fullchain.pem;
    ssl_certificate_key /etc/letsencrypt/live/dentaltripchina.com/privkey.pem;
    ssl_protocols TLSv1.2 TLSv1.3;
    ssl_ciphers HIGH:!aNULL:!MD5;

    # WordPress permalinks
    location / {
        try_files $uri $uri/ /index.php?$args;
    }

    location ~ \.php$ {
        fastcgi_pass unix:/run/php/php8.2-fpm.sock;
        fastcgi_index index.php;
        include fastcgi_params;
        fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
    }

    # Block access to sensitive files
    location ~ /\.ht { deny all; }
    location = /wp-config.php { deny all; }

    # Gzip compression
    gzip on;
    gzip_types text/css application/javascript image/svg+xml;

    # Browser caching for static assets
    location ~* \.(jpg|jpeg|png|gif|ico|css|js|woff2)$ {
        expires 1y;
        add_header Cache-Control "public, immutable";
    }
}

1.5 Cloudflare Setup (Recommended — Free Tier)
	•	Point nameservers to Cloudflare after DNS setup
	•	SSL/TLS mode: Full (strict) — requires valid SSL cert on server
	•	Enable: Auto Minify (HTML, CSS, JS), Brotli compression
	•	Page Rules: Cache Everything for static pages (homepage, procedure pages)
	•	Security: WAF — enable free ruleset, set sensitivity to Medium
	•	Speed: Enable Rocket Loader for JS (test first — can break some plugins)

cd /var/www/dentaltripchina/public

# Download WordPress

5.3 functions.php — Core Registrations

php — functions.php
<?php
// ── Enqueue assets ──────────────────────────────────────
function dtc_enqueue_assets() {
    wp_enqueue_style('dtc-main', get_stylesheet_directory_uri() . '/assets/css/main.css', [], '1.0.0');
    wp_enqueue_script('dtc-main', get_stylesheet_directory_uri() . '/assets/js/main.js', [], '1.0.0', true);
    // Pass AJAX URL to JS
    wp_localize_script('dtc-main', 'dtc_ajax', ['url' => admin_url('admin-ajax.php'), 'nonce' => wp_create_nonce('dtc_nonce')]);
}
add_action('wp_enqueue_scripts', 'dtc_enqueue_assets');

// ── Register Custom Post Types ───────────────────────────
function dtc_register_cpts() {

    // Procedure CPT
    register_post_type('procedure', [
        'labels'      => ['name' => 'Procedures', 'singular_name' => 'Procedure'],
        'public'      => true,
        'has_archive' => false,
        'rewrite'     => ['slug' => '', 'with_front' => false], // URLs at root: /dental-implants-china/
        'supports'    => ['title', 'editor', 'thumbnail', 'custom-fields', 'excerpt'],
        'show_in_rest' => true,
        'menu_icon'   => 'dashicons-heart',
    ]);

    // Hospital CPT
    register_post_type('hospital', [
        'labels'      => ['name' => 'Hospitals', 'singular_name' => 'Hospital'],
        'public'      => true,
        'has_archive' => false,
        'rewrite'     => ['slug' => 'hospital', 'with_front' => false],
        'supports'    => ['title', 'editor', 'thumbnail', 'custom-fields'],
        'show_in_rest' => true,
        'menu_icon'   => 'dashicons-building',
    ]);

    // City Guide CPT
    register_post_type('city', [
        'labels'      => ['name' => 'City Guides', 'singular_name' => 'City Guide'],
        'public'      => true,
        'has_archive' => false,
        'rewrite'     => ['slug' => '', 'with_front' => false], // Root-level URLs
        'supports'    => ['title', 'editor', 'thumbnail', 'custom-fields'],
        'show_in_rest' => true,
        'menu_icon'   => 'dashicons-location',
    ]);
}
add_action('init', 'dtc_register_cpts');

// ── Theme support ────────────────────────────────────────
function dtc_theme_setup() {
    add_theme_support('title-tag');
    add_theme_support('post-thumbnails');
    add_theme_support('html5', ['search-form', 'comment-form', 'gallery', 'caption']);
    register_nav_menus(['primary' => 'Primary Navigation', 'footer' => 'Footer Navigation']);
}
add_action('after_setup_theme', 'dtc_theme_setup');

// ── Disable XML-RPC (security) ───────────────────────────
add_filter('xmlrpc_enabled', '__return_false');

// ── Remove WordPress version from head (security) ────────
remove_action('wp_head', 'wp_generator');

Form Submission Behaviour
	•	On submit: validate all required fields client-side (JS) before AJAX POST to server
	•	On server: validate again server-side (never trust client-side only), check honeypot, verify nonce
	•	On success: replace form HTML with thank-you message — DO NOT redirect (kills GA4 goal tracking)
	•	Thank-you message: 'Thank you, [First Name]! We will respond to your WhatsApp or email within 2 hours. Check your spam folder if you don't hear from us.'
	•	Trigger: send email notification to operator (via WP Mail SMTP), send auto-reply to patient
	•	GA4 event: fire custom event 'inquiry_form_submit' with properties: procedure, country, city
	•	CRM: if HubSpot connected, create new Contact + Deal automatically

10.2 Image Optimisation Rules
	•	Format: WebP for all images. AVIF where browser support allows. Never BMP or TIFF.
	•	Hero images: max 200KB. Body images: max 100KB. Thumbnails: max 30KB.
	•	Dimensions: upload at max display size (never let WordPress shrink by 80%). Hero: 1920×1080. Card thumbnails: 800×600.
	•	Lazy loading: use loading='lazy' on ALL images below the fold. Hero image: loading='eager' + fetchpriority='high'.
	•	Alt text: required on every image. Descriptive, keyword-relevant where natural.
	•	Plugin: Smush or ShortPixel — configure to auto-convert to WebP and strip EXIF data.

10.3 CSS & JavaScript Rules
	•	No page builder — Gutenberg + Kadence Blocks only. No Elementor, Divi, or WPBakery.
	•	Minify CSS and JS via WP Rocket (combine and minify setting).
	•	Defer all non-critical JS: add defer or async attribute.
	•	Remove unused CSS: WP Rocket > Remove Unused CSS (enable, test carefully — can break things).
	•	Google Fonts: preconnect link in <head>, then load one font family only (Inter). Use display=swap.
	•	Third-party scripts (GA4, HubSpot): load via Google Tag Manager to centralise and defer.

11. Security Configuration
11.1 Wordfence Settings
	•	Firewall: Enable Web Application Firewall (WAF), mode: Learning then Extended Protection
	•	Login Security: Limit login attempts to 5. Lock out for 4 hours after 5 failures.
	•	Two-Factor Authentication: Enable for admin account (required).
	•	Login page CAPTCHA: Enable reCAPTCHA v3 on login form.
	•	Malware Scanner: Run weekly. Email alert on any detection.
	•	Block known bad IPs: Enable. Wordfence maintains a live threat IP feed.
